# LH 설계지침 검토 챗봇 V3

LH 설계지침(토목) 제3572호 HWP 원문을 텍스트 DB화하여 브라우저에서 검색하고 검토결과 및 조치계획 초안을 작성하는 GitHub Pages용 정적 웹앱입니다.

## 업로드 방법
1. 이 폴더의 `index.html`, `README.md`, `data/lh_db.json`을 GitHub 저장소 루트에 업로드합니다.
2. Settings > Pages > Deploy from a branch > main / root 저장
3. 생성된 GitHub Pages 주소를 직원들에게 공유합니다.

## 주의
- 내부 검토 보조용입니다. 최종 제출 전 HWP 원문에서 조항·표·페이지를 재확인하세요.
- 서버나 API 없이 실행되므로 직원들은 링크만 눌러 바로 사용할 수 있습니다.
