# LH 설계지침 검토 챗봇 V5

LH 설계지침(토목) 제3572호 전권 DB를 브라우저에서 검색하여, VE/기술심의/계약심사/일상감사 검토의견에 대한 관련 조항, 판단 요약, 조치계획 초안을 작성하는 정적 웹앱입니다.

## V5 개선사항

- 전 분야/전 공종 의미검색 가중치 적용
- 총칙, 가설, 토공, 구조물, 우수·오수, 상수·공동구, 옹벽, 도로·포장, 기타, 기초, 설계도서 작성기준 분류
- 단순 키워드가 아니라 공종 분류 + 전문 규칙 + 제목/본문 일치도를 함께 계산
- 관련도 낮은 조항 억지 출력 방지
- `저류지 바닥 포장`, `평균유출계수`, `오수관`, `상수관`, `비탈면`, `공사기간` 등 전 공종 검토의견 대응
